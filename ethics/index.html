2장 · 동양 윤리 — 유교 · 도가 · 불교

유교 ↔ 도가 대조 (도 → 성 → 덕 → 수양 → 지 → 이익 → 통치) · 불교 흐름
유교 ↔ 도가 대조 ★
유교 도가
도
하늘 — 천도 · 천리 · 천명 → 천인합일
우주 자연의 순리 — 스스로 그러함 (무위 · 자연) 무명 · 감각 초월 · 언어와 지식 초월
성
맹자 (성선) → 불인인지심 / 양지·양능 / 사단·사덕
⇒ 유지 · 보존 · 확충 · 실천 무력 통치 = 패도 ↔ 왕도 주장 순자 (성악) → 변화(교화)
(별도 성론 없음 — 자연 그대로의 본성을 회복)
덕
인(仁) → 인격체의 인간다움 / 도덕성 / 사랑
시비선악 분별옳고 그름·선악을 가린다. 무조건적 사랑이 아님
존비친소 분별 별애가까운 사람부터 차등적으로 사랑
↔ 겸애(묵가) — 차별 없이 똑같이 사랑
점진적 확대가족 → 이웃 → 사회로 넓혀 나감
효제 / 충서효제 = 부모께 효, 형제간 우애 (인의 출발점)
충서 → 추기급인(자기를 미루어 남에게 미침)
· 서(恕) 기소불욕 물시어인 — 내가 원치 않는 바를 남에게 베풀지 말라 (소극적) · 충(忠) 기욕립이립인 기욕달이달인 — 내가 서고자 하면 남을 세우고, 통달하고자 하면 남을 통달케 하라 (적극적)
예(禮) → 외면적 사회규범 → 비례물시·물청·물언·물동 → 예악 (예법과 음악)
유교의 인간 기준 · 인위적 노력을 비판
인위 · 작위적 ↓   무위 ↑ 무지 = 순수하고 소박한 상태 → 무욕 → 무위 ⇒ 겸허 · 겸손 → 부쟁(다투지 않음)
