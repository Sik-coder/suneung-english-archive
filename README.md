# 수능 영어 기출 아카이브 📘

2018학년도부터 최신까지, 수능 영어(외국어영역) **평가원 기출**을 한 곳에 모은 개인 학습 아카이브.
문제지 원본(PDF)은 저장하지 않고 **평가원·EBSi 다운로드 링크**로 연결하며, **듣기 음원은 유튜브 링크**로 엽니다.

👉 **대시보드:** [`index.html`](./index.html) — 회차별 링크 + 회독 체크 + 진도 관리

---

## 학습 플랜

컨설팅 기준 학습 루틴에 맞춰 구성했습니다.

- 하루에 **반 회분**씩: `18~30번` 풀고 채점 → `31~45번` 풀고 채점 → 답 이유 확인 + 모르는 단어만 정리
- 교재 없이 **평가원 기출만** (EBS 연계·교육청 모의 불필요)
- 목표: **2회독 = 54일**, 절대평가라 80점 이상(2~3등급)이면 충분

## 폴더 구조

```
suneung-english-archive/
├── index.html          ← 대시보드 (회차별 링크·회독 관리)
├── vocab/
│   └── index.html      ← 통합 단어장 (회차 필터로 골라 보기, 파일 1개)
└── explain/
    └── <학년도>-<회차>.html   ← 해석풀이 (회차당 지문 해석 + 문제 해설)
```

### 단어장에 단어 추가하기
`vocab/index.html`의 `WORDS` 배열에 회차 태그(`s`)를 붙여 줄만 추가하면 됩니다. 파일은 계속 하나.

```js
{ s:"2018-09", w:"단어", pos:"품사", m:"뜻", ex:"예문 <em>단어</em> ..." },
```

### 해석풀이 연결하기
해석 파일을 `explain/2018-06.html`처럼 넣고, `index.html`의 해당 회차에 `explain:true`를 추가하면
카드에 `📖 해석풀이` 버튼이 생깁니다.

```js
{ y:2018, m:"6", url:"...", vocab:"2018-06", explain:true },
```

## 링크 추가·수정하는 법

`index.html` 상단 `<script>` 안의 `SESSIONS` 배열에서 각 회차의 `url`을 수정합니다.
평가원 게시글 한 페이지에 **문제·정답·해설·듣기 음원**이 모두 들어 있어 링크는 회차당 하나면 됩니다.

```js
{ y:2018, m:"수능", url:"평가원_게시글_링크" },
```

- `url` — 평가원 기출 게시글 주소
- `flag` — (선택) 카드에 빨간 안내문구 표시. 예: `flag:"⚠ 링크 확인 필요"`
